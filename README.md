# Task Manager API

A RESTful API for managing tasks, built with Java and Spring Boot.

## Technologies

- Java 21
- Spring Boot
- PostgreSQL
- JPA / Hibernate

## Features (in progress)

- Create task
- List tasks
- Update task
- Delete task

## Getting Started

To run the project locally:

```bash
git clone https://github.com/your-username/task-manager-api.git
cd task-manager-api
./mvnw spring-boot:run
